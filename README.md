Démineur – Mode d’emploi rapide :

Le but du jeu est de révéler toutes les cases qui ne contiennent pas de mines, sans jamais cliqué sur une seule mine.
Et bien sur, pour les plus experts, il convient également de gagner le plus vite possible.

Chaque case révélée peut être vide ou contenir un chiffre allant de 0 à 8. 
Ce chiffre indique le nombre de mines présentes dans les 8 cases autour. 
Et c'est grace à cette information est au différents patterns qu'on peut en déduire l'emplacement des mines et donc progresser.

Commandes : 
- Clic gauche : révèle la case sélectionnée. 
_ Clic droit : place un drapeau sur une case pour signaler une mine ou enlève le drapeau si un est déjà présent.

Un drapeau est un moyen visuel et facultatif de marquer une mine. 
Il permet également d'empecher toute intéraction avec la case sur laquelle il est, évitant ainsi les potentielles missclick.

Le jeu se termine lorsque toutes les cases sûres ont été révélées (victoire) ou lorsqu’une mine est cliquée (défaite).
